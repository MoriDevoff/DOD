Где я? — запуск без Python
============================

1. Запустите файл WhereIAm.exe (двойной щелчок).
2. Дождитесь открытия браузера: http://127.0.0.1:8000/
3. Чтобы остановить игру — закройте процесс WhereIAm.exe в диспетчере задач
   или перезагрузите компьютер.

При первом запуске рядом с exe появятся папки media\ и файл db.sqlite3 —
это нормально (сохранение рекордов и загрузок).

Нужен интернет для карты OpenStreetMap и шрифтов.

Если не открывается сайт:
- проверьте, не занят ли порт 8000 другой программой;
- откройте http://127.0.0.1:8000/ вручную;
- посмотрите launcher_error.log рядом с WhereIAm.exe.

Файл WhereIAm.exe собирается разработчиком командой:
  my_django_project\build_onefile.bat
